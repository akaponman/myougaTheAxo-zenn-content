---
title: "はじめに"
---

Claude Codeは、Anthropicが提供するターミナルベースのAIコーディングアシスタントです。ただインストールして使うだけでも十分便利ですが、**適切に設定すると生産性が劇的に変わります。**

このガイドでは、以下の7つの領域を網羅します：

1. **CLAUDE.mdの設計パターン** — AIに的確な指示を出す「プロジェクト仕様書」
2. **設定ファイルの階層と権限** — 誰が何をできるかを制御する
3. **Hooksで自動化** — ツール実行の前後に自動処理を挟む
4. **MCPサーバー構築** — 外部ツールとの連携で可能性を拡張
5. **カスタムスキル（スラッシュコマンド）** — 再利用可能なプロンプトテンプレート
6. **チーム開発オーケストレーション** — 複数エージェントの協調
7. **実践ユースケース集** — 明日から使えるレシピ

公式ドキュメントには載っていない、現場で培った設定パターンを惜しみなく公開します。
